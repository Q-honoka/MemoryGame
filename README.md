# [神経衰弱] 

<img width="80%" height="80%" alt="game" src="https://github.com/user-attachments/assets/6582d36c-0e54-4c6d-b124-542147382b13" />

マウスで遊べるトランプゲームです。スペースキーでスタートして、ESCキーで終了します。
２枚のカードを選択して、それらが同じ数と色なら表向きになり、異なる場合は裏向きに戻ります。すべてのカードを表向きにしたらクリアです。

## 🎮 ゲーム概要
- **ジャンル**: トランプゲーム
- **プラットフォーム**: PC (Windows)
- **制作期間**: 約1か月
- **言語**: C++
- **使用ツール**: VisualStudio2026, DxLib

## 🌟 実装のポイント
- **CSVファイルの読み込み**: 52枚のトランプのデータを管理しやすくするためにCSVファイルのデータを読み込むようにしました。
- **トランプのカードデータの分離**:　カードのデータを「システム用」と「描画用」の２つに分けて管理しやすくしました。

## 💡 こだわった点 
- **２枚目選択後の待機時間の導入**: 待機時間にアニメーションを入れることで、入力の受け付けはされないがプログラムは止まっていないことを示すようにしました。
